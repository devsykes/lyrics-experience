# Sanity - Interactive Lyrics Experience

An immersive web-based musical journey featuring **"Sanity"** by Louis Tomlinson. This fan-made project focuses on the intersection of audio synchronization and aesthetic web design.

## Overview
The goal of this project is to create a "lyrics experience" where the interface isn't just a static display, but a living element that reacts to the music's rhythm and emotional beats.

## Key Features
* **Precise Synchronization:** Real-time lyric highlighting using JavaScript's Audio API.
* **Dynamic Atmosphere:** Visual states that transition between verses and the chorus.
* **Responsive Design:** A seamless experience across mobile and desktop devices.
* **Smooth Animations:** CSS-driven transitions for a premium feel.

## Tech Stack
- **HTML5:** Semantic structure and Audio management.
- **CSS3:** Custom properties (variables) and keyframe animations.
- **JavaScript:** Core logic for time-tracking and DOM manipulation.

## Tools & Environment
- **Editor:** Visual Studio Code
- **Extensions:** Live Server (for real-time preview)
- **Version Control:** Git & GitHub
